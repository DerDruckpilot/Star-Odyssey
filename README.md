# Star Odyssey

Browserbasiertes digitales Sci-Fi-Brettspiel mit TV-/Host-Ansicht und Smartphone-Controllern.

## Entwicklungsziel

Star Odyssey ist als statische Web-App/PWA umgesetzt und fuer TV-/Querformat sowie lokale Smartphone-Controller ausgelegt.

## Startanleitung

```bash
npm install
npm start
```

Danach den lokalen Browser-Link oeffnen, den `serve` ausgibt.
