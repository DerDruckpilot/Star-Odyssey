# Star Odyssey

Browserbasiertes Sci-Fi-/Space-Abenteuer als Webspiel-Prototyp.

## Entwicklungsziel

Star Odyssey wird zuerst als PC-Browser-Test aufgebaut. Spaeter kann optional eine Second-Screen-, TV- oder Fire-TV-Nutzung ergaenzt werden.

Die konkrete Spielmechanik wird in spaeteren Schritten erweitert.

## Startanleitung

```bash
npm install
npm start
```

Danach den lokalen Browser-Link oeffnen, den `serve` ausgibt.
